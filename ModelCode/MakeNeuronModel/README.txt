To create a dynamical cell model from the morphologies in ./MorphData (XX=[1,2,3,4,5,6]):

1) start ipython
2) type "execfile('makeCellBCXX.py')"
3) type "makeCell('mymodelname','./MorphData/BCXX.cell.nml')"

Worked for Neuron7.4, neuroml version 0.2.18, numpy version 1.11.0, python 2.7.12.
